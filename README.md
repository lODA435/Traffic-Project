  Проєкт: Система збору та логування даних про трафік
Flask-додаток, який автоматизує збір статистики про дорожні затори. 
Замість того, щоб копіювати дані вручну, програма сама звертається до API Google Maps для пошуку координат та до TomTom для отримання поточної швидкості машин.
  Основна задача системи - підготувати дані для аналізу. Під час запису програма відразу рахує відношення поточної швидкості до вільної, що показує реальний рівень 
завантаженості дороги. Також до кожного запису додається мітка часу (ранок, день, вечір або ніч), щоб потім можна було легко відсортувати дані за періодами доби.
Крім швидкості, система фіксує наявність ремонтних робіт чи аварій на ділянці. Всі результати зберігаються у звичайний CSV-файл. Це дозволяє швидко накопичити базу 
реальних спостережень, яку можна відкрити в Excel для побудови графіків або простого статистичного аналізу.
